# Картинки для игры «Курьер-раннер»

Положите сюда два файла, чтобы заменить пиксельную графику на свои изображения.

## Файлы

| Файл | Назначение | Рекомендуемый размер |
|------|------------|----------------------|
| **courier.png** | Моделька курьера (на самокате) | 40×50 px или 80×100 px (для Retina) |
| **logo.png** | Логотип компании, который курьер собирает в полёте | 20×20 px или 40×40 px (для Retina) |

## Имена и папка

- Папка: **`img/`** (в корне проекта, рядом с `index.html`).
- Курьер: **`img/courier.png`**.
- Значок для сбора: **`img/logo.png`**.

Если файлов нет или они не загрузились, игра по-прежнему рисует курьера и бонусы пиксельной графикой кодом.

## Формат

- Формат: **PNG** (с прозрачностью по желанию).
- У курьера ориентируйте персонажа вправо (в сторону движения).
- Логотип будет масштабироваться в квадрат 20×20 виртуальных пикселей на canvas.
