# Картинки для игры «Курьер-раннер»

Положите файлы в папку **`img/`** (в корне проекта, рядом с `index.html`).

## Фазы курьера

| Файл | Назначение | Когда показывается | Размер |
|------|------------|--------------------|--------|
| **courier_walk_1.png** | Пеший курьер, кадр 1 | 0–299 м | 40×50 px (или 80×100 для Retina) |
| **courier_walk_2.png** | Пеший курьер, кадр 2 | 0–299 м (анимация ходьбы) | 40×50 px |
| **courier.png** | Курьер на велосипеде | с 300 м до конца раунда | 40×50 px |

- **Пеший режим (0–299 м):** анимация из двух кадров — `courier_walk_1.png` и `courier_walk_2.png`.
- **С 300 м:** показывается одна моделька — `courier.png` (велосипед).

## Остальные файлы

| Файл | Назначение | Размер |
|------|------------|--------|
| **logo.png** | Логотип, который курьер собирает в полёте | 20×20 px (или 40×40 для Retina) |

Если каких-то файлов нет, игра дорисует курьера и бонусы пиксельной графикой кодом.

## Формат

- Формат: **PNG** (с прозрачностью по желанию).
- Курьера ориентируйте вправо (в сторону движения).
- Логотип масштабируется в квадрат 20×20 виртуальных пикселей на canvas.
